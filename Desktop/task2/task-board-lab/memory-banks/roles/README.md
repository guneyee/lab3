# Role Context

Use this directory for role-specific memory-bank files when the project needs specialized context.

Suggested files:
- [developer.md](./developer.md)
- [qa.md](./qa.md)
- [pm.md](./pm.md)

Each role file should capture:
- Primary responsibilities
- Decision boundaries
- Required artifacts and checklists
- Common prompts or recurring tasks
- Escalation paths
